Stridsregler för vapen

STY-krav

  • Vapnets STY-grupp 1 steg för hög → kan använda tvåhänt (om fästet tillåter) som separat färdighet
  • Vapnets STY-grupp 2+ steg för hög → CL halverad (kan tvåhänt ej hjälpa)
  • Vapen märkt 1–2H → enhand kräver STY-grupp 1 HÖGRE än angivet

Speciella vapen

Bola: 1. Slå anfallskast → träff 2. Slå FV-kast - Lyckas: avsett effekt (faller, arm fast, omtumlad) - Miss: 1T4 skada - 2 ben snodda → kan ej förflytta sig, springande varelse faller - 1 arm snodd → kan ej använda armen - Huvud snodat → 1T3 SR omtumling - Avlägsna: 1T3 SR

Lasso: 1. Slå anfallskast → träff 2. Slå FV-kast - Lyckas: avsett - Miss: ingenting - Offret övervinner STY + SMI / 2 (försvarar STY) för att bryta sig fri - Kring halsen/2 ben: anfallaren STY ×2

Oxpiska: - Fungerar som lasso MEN - Ger 1T2 skada oavsett FV-kast - Greppet varar bara den SR den träffar

Spjut mot springande: - Spjut/långspjut mot anstormande → vapnet sätts emot marken - Fungerar som vanlig attack under stridens FÖRSTA SR - Lyckas: normal skada + målets skadebonus (hästens SB om ryttare) - Från SR 2: normal strid

Lans/spjut från galopperande riddjur: - Kräver minst 15 m ansats - Använd hästens SB istf din - Om skada > målets STO: kastas 1T4 meter ur sadeln (fallskada)

Rustningsregler

Påsättning av rustning

  • Ostörd med hjälp: 2 minuter per rustningsdel
  • Utan hjälp: längre tid (⚠ exakt tid — se REG)
  • Fjällpansar och tyngre: klirrar → Smyga halveras

Varelser med STO 1–4

Kan enbart använda läderrustning som täcker hela kroppen. - Väger 5 BEP per STO - Abs = 2 (REG s.53 not 6)

Bevisade humanoider

Kan INTE ha tyngre rustning än läder på bröstkorg och mage. Vingar kan inte pansras.
