## Snabbreferens — Hur monster används

| Term | Förklaring |
|------|-----------|
| **KP** | Kroppspoäng = (STO + FYS) / 2 |
| **ABS** | Naturligt skydd (absorption) — fungerar som rustning |
| **Förfl.** | Rutor per SR (1 ruta = 1,5 m) |
| **FV** | Antal att slå under/lika med för att lyckas med anfallet |
| **Moral** | Stridsmoral (1T20 ≥ moral = behåller förnuftet) |
| **Grundmoral** | Utgångsvärde — modifieras av situation (REG s.62) |

**Moralslagets tidpunkter:**
- Ledaren stupar/flyr/ger sig/tas tillfånga
- Hälften av gruppen oförmögen att slåss
- Gruppen angrips överraskande
- Varelsen förlorar >50% KP
- Varelsen angrips av bärsärkande fiende med större STO

## Stridsmoral i praktiken

**Lyckat moralslag** → behåller förnuftet; fortsätter slåss eller retirar ordnat.

**Misslyckat moralslag:**
1. Kan ej gå bärsärk → flyr omedelbart
2. Kan gå bärsärk + >50% KP kvar → slår bärsärkskontroll
   - Lyckas → goes bärsärk (2 attacker/SR, ingen parering, förföljer flyende)
   - Misslyckas → flyr
3. Kan gå bärsärk + ≤50% KP → flyr

**Fummel vid moralslag** → ger upp helt, kastar vapnet.
**Perfekt moralslag** → slå bärsärkskontroll; om ej bärsärk = normalt lyckat slag.

**Omringad intelligent varelse:**
- Om >50% KP kvar: slår ny bärsärkskontroll; ej bärsärk = ger sig
- Om ≤50% KP: ger sig automatiskt

**Omringad ointelligent varelse:** slår sig fri.

## Bärsärk-effekter

En bärsärkande varelse/rollperson:
- Angriper vettlöst närmsta fiende
- Bryr sig ej om skador (kan förblöda)
- **2 attacker per SR** (den extra angreppet kommer sist i SR)
- **Parerar ALDRIG** (inte ens med sköld)
- När alla fiender flytt/slagits ur: slår ny bärsärkskontroll
  - Lyckas → förföljer flyende i FYS antal SR
  - Misslyckas → lugnar ner sig

**Frivillig bärsärk:**
- Stå stilla en hel SR och arbeta upp sig
- Slå 1T20 ≤ bärsärksvärde × 2
- Lyckas → goes bärsärk

## SL-notiser

### Hur man skapar monster med D&DE-systemet
1. Bestäm grundegenskaper (STY/FYS/STO/SMI/INT/PSY/KAR)
2. Beräkna KP = (STO + FYS) / 2
3. Beräkna Skadebonus från STY + STO
4. Bestäm vapens FV och skada

### Varelers STO och strid
- Varelser med stor skillnad i STO mot rollpersoner ger moralmodifikatorer
- STO-summa = summan av alla inblandades STO på en sida
- >33% skillnad = ±5 på moralslaget

#### Hamnskiftare *(Metamorphos voluntarius)*

*(Ingen standardiserad stat-block — hamnskiftare är humanoider som antar djurform)*

**Regler:** Hamnskiftare är humanoider av valfri ras med förmågan att förvandla sig till ett specifikt djur. Förvandlingen är äkta (inte illusion) — de erhåller djurets fysiska egenskaper (STY, STO, FYS, SMI) men behåller sin egna INT och alla INT-baserade färdigheter. Besvärjelser kan aldrig användas i djurform.

- **Vanligaste former:** Björn, rovfågel, svan, hjort, hund, katt, varg.
- **Ursprung:** Medfödd förmåga, gudomligt ingripande eller animism-ritualen DUOMORF PERMANENS.

**Källa:** MBK4, s. 60
