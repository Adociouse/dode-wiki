## Snabbreferens

### Grundmekanik
- Slå **1T20 ≤ FV (Färdighetsvärde)** för att lyckas
- **"1"** = Perfekt (slå igen: ≤ FV = perfekt slag)
- **"20"** = Fummel (slå igen: > FV = fummel)
- **CL** (Chans att Lyckas) = FV modifierat av situationen

### Stridssummering
1. Bestäm turordning: SMI (+1T6 valfritt) — högst agerar först
2. Anfallaren slår 1T20 ≤ FV = träff
3. Försvararen slår 1T20 ≤ FV = parering (om sköld/vapen)
4. Träff: slå vapenskada − motståndarens ABS = faktisk skada på KP
5. KP = 0: medvetslös/ur strid; KP = −FYS: dead

### KP-formel
**KP = (STO + FYS) / 2** (avrunda till närmaste heltal)
