# Drakar och Demoner Expert — Spelarviki

En regelreferens och uppslagsverk för **Drakar och Demoner Expert (1991)** och spelvärlden Ereb Altor.

Wikin är inte en ersättning för originalböckerna utan ett sökbart komplement — sammanfattningar, tabeller och korsreferenser som underlättar under pågående spel.

Live: https://adociouse.github.io/dode-wiki

---

## Innehåll

Grundegenskaper · Strid & Vapen · Färdigheter · Raser · Yrken · Magi · Utrustning · Hjältar · Varelser · Drakar · Demonologi · Källförteckning

---

## Originalmaterial

Originalböckerna finns skannade på Archive.org:
https://archive.org/details/drakar_och_demoner

Bok och sidnummer anges i wikin där det är möjligt. Om en uppgift är osäker eller svårläst markeras den med ⚠.

---

## Redaktionsprinciper

- Originalböcker är primärkällor. Egna sammanfattningar och förklaringar är välkomna — ordagranna boktexter är det inte.
- Officiella regler, tolkningar och husregler hålls isär och märks tydligt.
- Om två officiella källor motsäger varandra beskrivs skillnaden istället för att ena väljs bort.

---

## Bygga om från källtext

Kräver Python 3 med `markdown`-paketet:

```
pip install markdown
```

```
python make_wiki_multipage.py
```

HTML-filerna skrivs till `docs/`. Redigera aldrig `docs/` direkt — gör ändringar i `src/` och bygg om.
